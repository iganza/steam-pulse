* Key takeaways folded into the prompt
Smart defaults (never empty canvas), strict chart-type compatibility rules
(dual-axis only when units differ), progressive disclosure, perf budgets
(<2s render, ≤1000 points), colorblind-safe palette, full empty/loading/error
states, click-to-inspect, URL as source of truth.
