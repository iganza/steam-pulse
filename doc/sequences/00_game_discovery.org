   ✔  GameDiscoveredEvent published to: GameEventsTopic
   ✔  SNS subscription filter routes game-discovered → app-crawl-queue

The topic is GameEventsTopic — exactly right for this. SystemEventsTopic gets a catalog-refresh-complete at the end (summary
stats), but the per-game events go to GameEventsTopic so downstream consumers (future webhooks, analytics, etc.) can subscribe by
game lifecycle.
