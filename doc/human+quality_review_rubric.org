* Rubric

For each dimension, score 1-5 and write a one-line note. The note matters more
than the number — when you aggregate, the notes are what tell you what to fix.

** 1. Insight Sharpness

Score 1 = generic AI slop, 5 = specific and non-obvious.

Ask yourself: /if I showed this paragraph to a publisher who already knows this
genre, would they learn something? Or would they say "yes, obviously"?/

** 2. Evidence Discipline

Score 1 = hand-waves, 5 = every claim ties to a specific review, tag, or number.

Ask: /can I trace every strong claim back to a source? Or does the report
assert things without grounding?/

The automated checker verifies quotes are verbatim — this rubric question is
about whether the *reasoning* cites evidence, not just whether quotes exist.

** 3. Phase Coherence

Score 1 = phases feel disconnected, 5 = Phase 2/3 genuinely build on Phase 1.

Ask: /does Phase 3 reference and extend findings from Phase 1, or does each
phase restart from scratch?/

** 4. Uniqueness to This Game

Score 1 = could be reskinned for any deckbuilder, 5 = unmistakably about THIS
game.

*Swap test*: mentally replace the game name with a similar game's name (e.g.
swap "Slay the Spire" for "Monster Train"). Does the report still make sense?
If yes, that's bad — the report isn't doing game-specific work.

** 5. Saleability

Score 1 = no one pays for this, 5 = a publisher would pay $49-$499 for this
report.

Gut check. Trust your instinct. This is the dimension that maps most directly
to the hybrid business model (reports-led).

* Decision Gate

** Greenlight criteria (ALL must hold)

- [ ] Overall average ≥ 3.5
- [ ] No individual dimension averages < 3.0
- [ ] No stratum averages < 2.5
- [ ] No single failure mode appears in >30% of reports
- [ ] Automated checker shows zero metadata fact-check failures on the sample

** If greenlit

Proceed with Phase 1-3 on the remaining 116 games in the wedge. Then run Phase
4 (cross-genre synthesis) once on the full 141. Budget check: ~$116 + Phase 4
cost remaining against ~$500 starting budget.

** If blocked

Identify the weakest prompt (lowest-scoring phase × dimension combo). Patch
that prompt. Re-run the sample — NOT the whole wedge — and re-score with this
rubric. Iterate until greenlit.

Remember: the stratified 25 cost ~$25. Re-running it is a cheap iteration
loop. Running the full 141 on a broken prompt is not.

* Appendix: Fast-Mode Review (optional)

If you want a cheaper first pass before the full rubric, do this "fast mode"
in ~3 min per report:

1. Read only the =one_liner= and =design_strengths=
2. Swap test (dimension 4) — does it feel like this game or any game?
3. If it fails the swap test, flag the whole report and move on — no need
   to score the other dimensions
4. If it passes, add to the "do the full rubric" pile

Useful when you suspect a broad quality issue and want signal fast.
