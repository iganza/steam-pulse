* Key Takeaways (applicable to all UI work)
- 5-7 items per visible filter group before "Show more" truncation (NN/g)
- Popularity sort > alphabetical sort for filter values (NN/g)
- Counts next to every filter value — critical for user confidence (Baymard)
- Mega menus >50 links see 34% higher bounce (Baymard) — users default to search
- Accordions: good when users need 1-2 sections, bad when they need most/all
- Tag clouds: consistently ignored by users — do not use
- Always provide type-ahead search alongside category browsing for large taxonomies
