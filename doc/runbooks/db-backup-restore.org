#+TITLE: Local Database Backup & Restore — Runbook
#+DESCRIPTION: Local Postgres backup/restore commands for SteamPulse dev DB.

* Backup
Runs =pg_dump= inside the Docker container — no version mismatch:

#+begin_src bash
./scripts/backup_db.sh
# Output: backups/steampulse_YYYYMMDD_HHMMSS.sql.gz
#+end_src

* Restore
From a backup file:

#+begin_src bash
gunzip -c backups/steampulse_TIMESTAMP.sql.gz | docker exec -i steam-pulse-db-1 psql -U steampulse -d steampulse
#+end_src

* Test database
Tests use a separate database — never the dev db:

#+begin_src bash
TEST_DATABASE_URL=postgresql://postgres:postgres@localhost:5432/steampulse_test poetry run pytest tests/ -q
#+end_src
