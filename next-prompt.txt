  Option A — Run locally pointed at staging AWS (simplest):

   DATABASE_URL="postgresql://..." \
   AWS_DEFAULT_REGION=us-west-2 \
   poetry run python scripts/seed.py --limit 50

  But you'd need the RDS connection string, which requires either a bastion host or RDS Proxy to reach it from your laptop.

  Option B — Run as a one-off Lambda invocation (correct way):

   aws lambda invoke \
     --function-name $(aws lambda list-functions --query "Functions[?contains(FunctionName,'Seed')].FunctionName" --output text) \
     --payload '{"limit": 50}' \
     --region us-west-2 \
     response.json && cat response.json
