╔══════════════════════════════════════════════════════════════════════╗
║                     KEY ROUTING RULES (CloudFront)                   ║
╚══════════════════════════════════════════════════════════════════════╝

  CloudFront behaviours (in precedence order):

  /api/*          ──────────────────────────►  ApiFn  (FastAPI Lambda URL)
  /_next/static/* ─────────────────────────►  S3  (static assets)
  /static/*       ─────────────────────────►  S3
  /*              ──────────────────────────►  FrontendFn  (Next.js Lambda URL)

║                    ENV VAR RESOLUTION                                ║

  FrontendFn (Next.js Lambda)
    API_URL = https://abc123.lambda-url.region.on.aws/   ← set by CDK
        │
        └── used only server-side (SSR)
            never exposed to browser

  Browser JS
    NEXT_PUBLIC_API_URL = ""   (same-origin, CloudFront handles routing)
        │
        └── fetch("" + "/api/games") = fetch("/api/games")
            CloudFront routes /api/* → ApiFn
